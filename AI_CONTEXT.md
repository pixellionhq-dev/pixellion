PROJECT: PIXELLION

Pixellion is a competitive digital advertising platform where brands buy pixels on a global grid and place their logos.

The canvas contains 1,000,000 pixels. Brands can purchase pixel blocks and upload their logo which appears on the board.

Users can zoom, drag, and explore the canvas. Each pixel block links to the brand's website.

TECH STACK

Frontend:
React / Next.js
Tailwind CSS

Backend:
NestJS API

Database:
PostgreSQL with Prisma

Hosting:
Frontend: Vercel
Backend: Render

MAIN FEATURES

Pixel Canvas
Brands buy pixels on a large grid.

Logo Upload
Brands upload their logo which appears on the board.

Leaderboard
Shows top brands by pixel ownership.

Brand Directory
List of all brands that purchased pixels.

Authentication
Users create accounts and manage purchases.

Payments
Users pay to purchase pixels.

PROJECT GOAL

Pixellion should feel extremely smooth and fast like top websites such as Apple, Google, and Instagram.

The board should render quickly even with thousands of logos.

Images must load instantly for all users across devices.
